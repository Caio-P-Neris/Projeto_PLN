# Projeto_PLN
Projeto feito por: Caio Pereira Neris, RA 11202230288; Mateus Batistelli Vignola, RA: 11202230362

Neste trabalho, implementamos uma ferramenta de análisa do mercado financeiro por meio de notícias (usando a NewsAPI), possibilitando:

  Análise de sentimentos:

  da economia brasileira,
  sua comparação com a economia de outros países,
  de empresas, setores e ações específicas

  Classificação de relevância e similaridade das notícias para cenário econômico, extraindo palavras-chaves

  Sumarização do cenário econômico

  Sistema de perguntas e respostas sobre a economia
